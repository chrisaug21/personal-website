CSS Variables

Include these CSS variables in your root stylesheet to easily implement the color scheme across your website:

/* Color Palette */ :root { --blue-primary: #0077CC; --blue-secondary: #B0C4E0; --gold-accent: #F2C94C; --white: #F8F9FA; --black: #2F4858; --gray-light: #E9ECEF; --gray-medium: #ADB5BD; --gray-dark: #495057; }

Color Usage Guidelines

  • Primary Blue (#0077CC): Use for main navigation, primary buttons, headings, and key UI elements.
  • Periwinkle Blue (#B0C4E0): Use for secondary elements, supporting UI, card headers, and secondary buttons.
  • Gold (#F2C94C): Use sparingly for calls-to-action, feature highlights, and important UI elements you want to emphasize.
  • Off-White (#F8F9FA): Use for page backgrounds and content areas.
  • Dark Blue-Gray (#2F4858): Use for text, icons, and other content that needs high contrast.

Usage Guidelines

Follow these guidelines to maintain a consistent brand appearance across your website and digital presence.

General Guidelines

  • Always use the defined color palette and typography system
  • Maintain proper spacing and alignment across all elements
  • Ensure responsive behavior across different screen sizes
  • Use consistent iconography that matches the brand's professional style
  • Apply hover states to interactive elements to enhance usability

Color Usage

  • Use Ocean Blue (#0077CC) as the primary brand color for main UI elements, important headings, and primary call-to-action buttons
  • Use Periwinkle Blue (#B0C4E0) for secondary UI elements, supporting information, and less prominent interactive elements
  • Use Gold (#F2C94C) as an accent color to highlight important information or to create visual interest - use sparingly for maximum impact
  • Use the neutral colors for backgrounds, text, and UI frameworks
  • Ensure proper contrast between text and background colors for optimal readability

Typography Guidelines

  • Use Nunito for all headings with appropriate hierarchy
  • Use Lato for all body text, UI elements, and buttons
  • Maintain consistent type scaling throughout the website
  • Use 800 weight for H1 headings and 700 weight for all other headings
  • Use 400 weight for body text and 700 weight for emphasized text

Component Usage

  • Use cards to group related content and provide clear visual hierarchy
  • Use feature cards to highlight key professional strengths or services
  • Use pills/tags to organize and display skills, expertise areas, and categories
  • Use consistent button styles based on their importance and function
  • Apply the timeline component to visualize professional growth and career progression
  • Use contact cards to provide clear communication channels
